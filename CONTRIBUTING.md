## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/YOUR_USERNAME/satset-obsidian-sync.git`
3. Install dependencies: `npm install`
4. Build: `npm run build`
5. For development with auto-rebuild: `npm run dev`

## Testing

1. Copy `main.js` and `manifest.json` to your Obsidian vault's `.obsidian/plugins/satset-sync/` directory
2. Reload Obsidian (Ctrl+R) to test changes

## Pull Requests

- Keep changes focused and minimal
- Update README if adding new features
- Test your changes on a fresh vault before submitting

## Code Style

- Use TypeScript strict mode
- Follow existing naming conventions
- Add JSDoc comments for public methods

## Reporting Issues

When reporting bugs, please include:

- Obsidian version
- Plugin version
- Steps to reproduce
- Any error messages from the developer console (Ctrl+Shift+I)
